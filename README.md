# Fraud Detection in Decentralized Finance (DeFi) Transactions Using ML and Smart Contracts

## Authors

* Harsh Rudrawar
* Sumitra A. Jakhete
* Dhanashree Somani

## Research Overview

This research presents a hybrid framework for real-time fraud detection in Decentralized Finance (DeFi) ecosystems.

The framework combines:

* Off-chain Machine Learning models
* Chainlink Decentralized Oracle Networks
* On-chain Smart Contract enforcement

to proactively detect and mitigate fraudulent blockchain activity.

---

## Key Contributions

* Real-time wallet risk scoring
* Flash loan attack detection
* Automated token blacklisting
* Chainlink oracle integration
* Smart-contract-based mitigation
* Low-latency fraud response framework

---

## Architecture

The framework consists of:

### Off-Chain Layer

* Data preprocessing
* Feature engineering
* Fraud prediction models
* Continuous retraining pipeline

### Oracle Layer

* Chainlink Decentralized Oracle Network (DON)
* Secure risk-score transmission

### On-Chain Layer

* Smart contract monitoring
* Automated enforcement
* Transaction blocking and mitigation
